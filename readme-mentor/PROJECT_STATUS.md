## 🎯 Usage Instructions

### For Developers
1. **Clone the repository**
2. **Install dependencies**: `npm install`
3. **Configure environment**: Copy `.env.example` to `.env.local`
4. **Setup Supabase**: Create project and run SQL schema
5. **Add API keys**: OpenRouter, Gemini, GitHub (optional)
6. **Start development**: `npm run dev`

### For End Users
1. **Visit the application** (when deployed)
2. **Paste GitHub repository URL**
3. **Wait for AI analysis** (30-60 seconds)
4. **Review generated README**
5. **Copy or download** the markdown file
